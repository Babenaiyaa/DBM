# Digital Tailor Shop Web Platform:   Seniour Cuts

This project is a web-based platform developed for an existing home-based tailor shop. It allows customers to browse designs, place custom orders, track order progress, and communicate directly with the tailor, while providing the admin with tools to manage products, orders, and analytics.

## Features

**Customer-Facing:**
- Browse ready-made products and new arrivals
- Place custom orders with measurements and design references
- Add items to cart and track order progress
- View promotions, discounts, and package deals
- Submit reviews and like favorite designs

**Admin-Facing:**
- Dashboard to manage orders, customers, and inventory
- Add, edit, or remove products
- Upload blog posts and promotional content
- Track order workflow and generate basic analytics

**Planned Enhancements:**
- Social media integration (Facebook, Instagram)
- SMS notifications for order updates

## Technology Stack

- **Frontend:** React, TypeScript, HTML/CSS
- **Backend:** Node.js, Express.js, REST APIs
- **Database:** MongoDB with Mongoose ODM
- **Authentication & Security:** JWT, bcrypt
